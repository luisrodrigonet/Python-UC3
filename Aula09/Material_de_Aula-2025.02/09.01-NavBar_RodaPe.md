# 09.01 - Implementação de Navbar e Rodapé com Bootstrap
---

**Pesquisa e Implementação:**
Pesquisem a documentação oficial do **Bootstrap** para criar e utilizar os componentes de navegação no projeto.

**Links Úteis:**

1.  **Navbar (Barra de Navegação):** Busque a documentação sobre a criação e uso de *navbars* para a navegação estruturada entre as páginas.
    * [Documentação Navbar - Bootstrap v5.3](https://getbootstrap.com/docs/5.3/components/navbar/)

2.  **Rodapé (*Footer*):** Consulte os exemplos para estruturar um rodapé para declarações extras e informações da empresa.
    * [Exemplos de Footers - Bootstrap v5.3](https://getbootstrap.com/docs/5.3/examples/footers/)

**Objetivos da Aula:**
O foco principal é aprimorar a declaração da navegação do projeto utilizando o Bootstrap para maximizar a clareza e a fluidez:

1.  **Navegação (*Navbar*):** Melhorar a forma de navegação entre as páginas utilizando o componente *navbar* do Bootstrap.
2.  **Rodapé (*Footer*):** Criar um rodapé para incluir declarações extras e informações sobre a empresa. Os alunos devem verificar quais opções e dados (como informações de contato ou direitos autorais) serão inseridos nesta área do site.
