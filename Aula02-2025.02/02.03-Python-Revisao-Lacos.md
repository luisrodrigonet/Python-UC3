## :one: La√ßo For

Um la√ßo **`for`** executa um bloco de c√≥digo repetidamente para cada item em uma sequ√™ncia. Ele automatiza tarefas que precisariam ser feitas manualmente para cada item, economizando tempo e esfor√ßo de programa√ß√£o.

A sintaxe b√°sica √© a seguinte:


  - **`for`**: palavra-chave que inicia o la√ßo.
  - **`item`**: uma vari√°vel tempor√°ria que assume o valor de cada elemento da `sequ√™ncia` a cada itera√ß√£o. Voc√™ pode dar qualquer nome a essa vari√°vel, mas o ideal √© que seja algo descritivo.
  - **`in`**: palavra-chave que indica que o la√ßo vai iterar sobre a `sequ√™ncia`.
  - **`sequencia`**: o objeto iter√°vel sobre o qual o la√ßo ir√° percorrer.

## :two: La√ßo While

O la√ßo **`while`** executa um bloco de c√≥digo repetidamente **enquanto** uma condi√ß√£o espec√≠fica for verdadeira. 

A l√≥gica √© simples: o Python avalia a condi√ß√£o, e se ela for **`True`**, o bloco de c√≥digo √© executado. Ap√≥s a execu√ß√£o, a condi√ß√£o √© verificada novamente. Esse ciclo se repete at√© que a condi√ß√£o se torne **`False`**.

**Sintaxe B√°sica**

```python

    # Bloco de c√≥digo a ser executado
    # Este bloco deve conter algo que, eventualmente,
    # far√° a condi√ß√£o se tornar False para evitar um loop infinito
```

  - **`while`**: A palavra-chave que inicia o la√ßo.
  - **`condicao`**: Uma express√£o que retorna `True` ou `False`.
  - **`:`**: Dois pontos que indicam o in√≠cio do bloco de c√≥digo.
  - **Bloco de c√≥digo**: O c√≥digo indentado que ser√° executado repetidamente.


### ü¶ï **Exemplo Pr√°tico: Contagem Progressiva**

Vamos criar um programa simples que conta de 1 a 5.

```python

    print(f"O contador agora √©: {contador}")
    contador += 1  # Incrementamos o contador para que a condi√ß√£o mude

```

**An√°lise do C√≥digo:**

1.  **`contador = 1`**: A vari√°vel `contador` √© inicializada com o valor `1`.
2.  **`while contador <= 5:`**: O Python verifica se `1` √© menor ou igual a `5`. Como √© `True`, o la√ßo come√ßa.
3.  **`print(...)`**: A mensagem √© exibida.
4.  **`contador += 1`**: O valor de `contador` √© atualizado para `2`.
5.  O Python retorna para o in√≠cio do `while` e verifica a condi√ß√£o novamente. O processo se repete at√© que `contador` se torne `6`.
6.  Quando `contador` √© `6`, a condi√ß√£o `6 <= 5` √© `False`, e o la√ßo √© encerrado. O programa continua a execu√ß√£o na linha seguinte.



### üëª **Cuidado com o La√ßo Infinito\!**

O erro mais comum ao usar `while` √© esquecer de atualizar a vari√°vel de controle dentro do la√ßo, o que impede que a condi√ß√£o se torne `False`. Isso causa um **la√ßo infinito**, onde o programa executa indefinidamente.

üíÄ **Exemplo de La√ßo Infinito:**

```python

    print(f"O contador √©: {contador}")
    # Esquecemos de adicionar 'contador += 1'
    # Este la√ßo rodar√° para sempre!
```

üôã **Observa√ß√£o:** Se seu programa travar, voc√™ pode interromper a execu√ß√£o pressionando `Ctrl + C` no terminal.
